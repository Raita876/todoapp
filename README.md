# TODO Application

ポートフォリオとして公開しているタスク管理 API。主に以下のアーキテクチャおよびツールを使用。

- 設計
  - ドメイン駆動開発
  - Rest API

- 言語
  - Go

- ライブラリ
  - gin
  - gorm

- データベース
  - postgreSQL

- テスト
  - gotests
  - runn

- ドキュメント
  - swagger

- CI/CD
  - goreleaser
  
## API

- <http://localhost:8080/ping>
- <http://localhost:8080/api/v1/tasks>
- <http://localhost:8080/swagger/index.html>
